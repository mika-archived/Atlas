# Lambda Computing Tasks

AWS Lambda で実行される JavaScript (Node.js) ファイルです。  
フォルダ名でリソースを、ファイル名でメソッドを示しています。

メソッド名は Rails の Resource Routing に沿っています。


| Method        | File name    |
| ------------- | ------------ |
| `GET /`       | `index.js`   |
| `GET /:id`    | `show.js`    |
| `POST /`      | `create.js`  |
| `PATCH /:id`  | `update.js`  |
| `DELETE /:id` | `destroy.js` |
